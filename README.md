# Auro

Auro is a modern web application developed using Next.js, JavaScript, Tailwind CSS, and Prisma. The project was created to practice building structured web applications with reusable components, clean styling, and database-ready architecture.

## Features

* Next.js-based project structure
* Reusable frontend components
* Tailwind CSS styling
* Prisma setup for database integration
* Organized source folder structure
* Modern JavaScript development workflow

## Technologies Used

* Next.js
* JavaScript
* Tailwind CSS
* Prisma
* CSS

## Purpose

This project was developed to improve my skills in modern web application development, frontend structure, styling, and database integration using Prisma.
